# Contributing

This repository is primarily a documentation project for the **HALO TOUCH USB Hub Dock with Clock V2**.

Contributions should improve the quality, accuracy, and usefulness of the public documentation.

## Good Contributions

- Corrections to technical specifications
- Clearer setup or operation steps
- Better troubleshooting guidance
- Verified notes about firmware behavior or feature support
- Additional public reference links, reviews, or tutorials
- Photos, diagrams, or translated documentation with clear source context

## Source Quality

When adding technical information, prefer:

- official product pages
- original manuals or setup articles from the product author
- direct hardware or firmware release notes
- first-hand testing notes that clearly state firmware version and setup conditions

If a claim is uncertain, mark it clearly instead of presenting it as confirmed fact.

## Pull Request Guidelines

1. Keep each pull request focused.
2. Use plain, factual English.
3. Preserve source links where possible.
4. Separate verified specifications from assumptions or interpretation.
5. Include the source used for any technical correction.

## Issues

When reporting a problem, include:

- firmware version
- power configuration
- TF card size and format
- host OS
- active feature or mode
- steps to reproduce the issue
