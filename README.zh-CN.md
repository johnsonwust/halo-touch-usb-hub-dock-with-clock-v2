## 产品简介

HALO TOUCH V2 是一个桌面多功能扩展坞，集成了：

- USB 2.0 HUB
- 圆形 IPS 显示屏
- 触控交互
- 旋钮控制
- TF 卡读卡器
- 电压 / 电流 / 功率监测
- 番茄钟
- MP3 音乐盒
- 相册 / 动画播放
- 麦克风频谱显示
- AIDA64 电脑信息显示
- Windows Surface Dial 风格控制

## 主要接口

- `USB-PC`：连接电脑的 USB 2.0 Type-C 上行口
- `PWR-PD`：额外 PD 供电输入
- `USB1` / `USB2`：USB 2.0 Type-A 下行口
- `USB3`：USB 2.0 Type-C 下行口
- `TF`：TF / microSD 卡槽
- `LAN`：100M 网口

## 供电说明

- 默认可以通过 `USB-PC` 由电脑供电
- 负载较高时可通过 `PWR-PD` 接入 PD 电源
- 不要把 USB-C 充电器直接插到 `USB-PC` 口来代替 `PWR-PD`

## TF 卡目录

原始说明中提到常见目录包括：

- `fonts`
- `mjpeg`
- `music`
- `night7`
- `pic`
- `aida64`

替换 TF 卡时建议：

- 容量不大于 32GB
- 格式为 FAT32
- 不要删除设备运行所需的系统文件

## Wi-Fi 设置

- 仅支持 2.4GHz Wi-Fi
- 默认热点名：`My-Ap`
- 默认密码：`12345678`
- 配网页面：`192.168.4.1`

## 常见功能

- USB 端口电压 / 电流 / 功率监测
- 25 分钟默认番茄钟
- `music` 目录 MP3 播放
- `mjpeg` 动画播放
- `pic` 目录 JPG 图片播放
- 麦克风频谱显示
- AIDA64 系统监视器显示
- 固件 `1.3.0+` 支持 Surface Dial 风格功能
- 系统菜单支持 OTA 更新
